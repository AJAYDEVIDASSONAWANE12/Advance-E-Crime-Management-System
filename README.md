# Advance-E-Crime-Management-System
 1....Crime Matrix is a secure, web-based police management system designed to digitize crime registration, case assignment, and public tracking. It streamlines communication between the Station House Officer (SHO), CID Investigators, and the Admin, while allowing the public to track case status using a unique Crime ID.
